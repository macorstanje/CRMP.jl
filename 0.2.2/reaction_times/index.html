CRMP.decreasing_rate — Type
decreasing_rate <: method

used when the rates of the reactions in the network are decreasing in time. This method utilizes a Poisson thinning with the current state as upper bound

source
CRMP.increasing_rate — Type
increasing_rate <: method

used when the rates of the reactions in the network are increasing in time This method utilizes a Poisson thinning with the time at t+δ as upper bound.

source
CRMP.thinning — Type
thinning <: method

Can be used when the rates of the reactions in the network are bounded. Utilizes the bounds in lemma ... (lower bound on h̃(t,x), upper bound on h̃(t,x+ℓ.ξ))

CRMP.gettime — Function
gettime(::constant_rate, ℓ::reaction{T}, t::Real, x::Union{T, Array{T,1}}) where {T<:Real}

Returns an Exp(ℓ.λ(t,x))-random variable, provided ℓ.λ(t,x)>0. Else returns 1e10

source
gettime(::decreasing_rate, ℓ::reaction, t, x::Union{T, Array{T,1}}) where {T<:Real}

Returns the reaction time of reaction ℓ if ℓ.λ is decreasing in time. Uses a thinning algorithm with upper bound ℓ.λ(t,x). Returns 1e10 when ℓ.λ(t,x)=0.

source
gettime(::increasing_rate, ℓ::reaction{T}, t, x::Union{T, Array{T,1}}, setδ::Function) where {T<:Real}

Returns the reaction time of reaction ℓ if ℓ.λ is increasing in time. Uses a thinning algorithm on compact intervals [t, t+δ] where δ is obtained from setδ and time by δ is moved if the reaction time is not below δ.

source
CRMP.setδ — Function
setδ(η::Real, diff::Float64, obs::observation{T}) where {T}

returns the function $(ℓ, t, x) ↦ T - t - 1/( 2*log(η)/diff + 1/(T-t) )$ for the thinning algorithm.
