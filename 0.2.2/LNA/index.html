CRMP.LNAR — Type
LNAR

LNA method with restart, guided rates are determined through solving a system of ODEs at each reaction time. julia-repl julia> obs = partial_observation(1.0, [2, 5], [1 0 0 ; 0 1 0], 0.05) julia> LNAR(obs, GTT(1.0, 1.0, 1.0, 1.0))`

source
CRMP.LNAR_death — Type
LNAR_death

LNA method with restart, closed form solution to the ODEs specifically implemented for the death process. Works the same as LNAR

source
