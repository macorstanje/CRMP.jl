CRMP.dist² — Function
dist²(ϕ)

Given a latent variable ϕ (possible matrix-valued), returns a quadratic distance function on 𝕊. For now, we keep it as dist²(ϕ)(x,y)=|y-x|^2/ϕ .

CRMP.C — Function
C(ℓ::reaction{T}, x::Union{T, Array{T,1}}, xT::Union{T, Array{T,1}}, d²::Function) where {T<:Real}

Returns the sign of reaction 'ℓ' at 'x' given a squared distance function 'd²' and a desired endpoint xT. Returns d²(xT, x+ℓ.ξ) - d²(xT,x)

CRMP.condition_reaction — Function
condition_reaction(ℓ::reaction{S}, xT::Union{S, Array{S,1}}, T::Real, d²::Function) where {S<:Real}

Given a reaction 'ℓ', a desired endpoint 'xT', end time 'T' and squared distance function 'd²', returns a reaction with the same difference vector ξ, but with the conditioned rate specified in the paper.
