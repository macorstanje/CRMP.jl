CRMP.ChemicalReactionProcess ‚Äî Type
ChemicalReactionProcess{T<:Real}

A chemical reaction process is set-up as a tuple of a vector of species ùíÆ with 'String's as input and a vector of 'reaction{T}'s. I still need some way to distinct between process and network For example, a poisson process is set-up as follows

julia> plus1 = reaction((t,x) -> x, 1)
julia> PoissonProcecss = ChemicalReactionProcess(["Counts"], [plus1])

Alternatively, if there is just one reaction, or species, one could omit the 'Array'.
