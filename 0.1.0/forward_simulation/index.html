Forward simulation

Forward simulation of chemical reaction processes in various settings

CRMP.next_jump — Function
next_jump(::constant_rate, t::Real, x::Union{T, Array{T,1}}, P::ChemicalReactionProcess{T}) where {T<:Real}

If the process is of constant (in time) rate, simulates exponential jump times for all reactions and returns the reaction with the shortest jump time along with the respective time.
