gettime(::constant_rate, ℓ::reaction{T}, t::Real, x::Union{T, Array{T,1}}) where {T<:Real}

source
gettime(::decreasing_rate, ℓ::reaction, t, x::Union{T, Array{T,1}}) where {T<:Real}

Returns the reaction time of reaction ℓ if ℓ.λ is decreasing in time. Uses a thinning algorithm with upper bound 'ℓ.λ(t,x)'

source
gettime(::increasing_rate, ℓ::reaction{T}, t, x::Union{T, Array{T,1}}, setδ::Function) where {T<:Real}

Returns the reaction time of reaction ℓ if ℓ.λ is increasing in time. Uses a thinning algorithm on compact intervals [t, t+δ] where δ is obtained from 'setδ' and time by δ is moved if the reaction time is not below δ.
